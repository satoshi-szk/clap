# エクステンション ID

## 命名

エクステンションは次の形式で命名される：`clap.$NAME/$REV`。
ここで：
- `$NAME` はエクステンションの名前。
- `$REV` はエクステンションのリビジョン。これは各イテレーションで増加する整数。1から始める必要がある。

サードパーティによって作成され、CLAPプロジェクトによって公式に公開されていないエクステンションについては、次の形式を使用してください：`$REVERSE_URI.$NAME/$REV`。
ここで：
- `$REVERSE_URI` は `com.bitwig` のようなもの。

## ドラフト

エクステンションが[draft](../include/clap/ext/draft/)フォルダにある場合、ドラフトエクステンションとみなされる。
[all_ja.h](../include/clap/all_ja.h)にも含めることを確認する。

エクステンションがドラフトから安定版に移行する際、そのエクステンションIDは変更してはならない。
[all_ja.h](../include/clap/all_ja.h)からのinclusionを[clap_ja.h](../include/clap/clap_ja.h)に移動する。

すべてのエクステンションはまずドラフト段階を経る必要がある。

## ファクトリー ID について

エクステンション ID について述べたことは、ファクトリー ID にも対称的に適用される。

## 履歴

このドキュメントが書かれたバージョン1.2.0以前では、既存のエクステンションはこれらのルールに従っていなかった。
私たちは互換性を破ることなく一部のドラフトエクステンションを安定化したかったが、それらのエクステンション ID には文字列 `draft` が含まれていた。
これらの文字列はユーザーに見えるものではなかったが、それでも削除したかったので、このドキュメントに従ってエクステンション ID を更新し、ドラフトバージョンとの後方互換性を提供するために `_COMPAT` サフィックス付きの ID を導入した。
